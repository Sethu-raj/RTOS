# STM32 FreeRTOS Projects

This repository contains FreeRTOS-based projects developed on STM32F407 Discovery Board using STM32CubeIDE.

## Topics Covered

- Task Creation
- Task Deletion
- Task Delay
- Task Priority
- Task Notification
- Idle Task Hook
- Queue Management
- Binary Semaphore
- Counting Semaphore
- Mutex
- Stream Buffer
- Message Buffer

## Hardware

- STM32F407VG Discovery Board
- ARM Cortex-M4
- Onboard LEDs and User Button

## Software

- STM32CubeIDE
- STM32CubeMX
- FreeRTOS Kernel

## Project Structure

## Features

- Multitasking
- Inter-task Communication
- Synchronization Mechanisms
- Real-Time Scheduling
- Interrupt Integration
